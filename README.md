# PocketJS

PocketJS is a tiny mobile interaction framework for native-feeling web apps.

It focuses on mobile patterns developers rebuild again and again:

- iOS-style bottom sheets
- swipe cards
- gesture handling
- native-feeling motion
- mobile-first UI helpers

## Run locally

```bash
npm install
npm run dev
```

## Current demo

The current demo shows:

```js
Pocket.createBottomSheet()
Pocket.createSwipeCards()
```

## Vision

PocketJS is not trying to replace React, Vue, or Svelte.

It is a mobile interaction layer for the web.
